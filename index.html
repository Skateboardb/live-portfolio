PROJECTS

FilmBase

A collaborative effort, FilmBase was created out of a mutual love for cinema and a need to incorporate a number of elements into a bootcamp project including an unfamiliar API and a UI that modeled a popular site. When a movie is searched, the user is presented with info in a way that somewhat mirrors the UI on IMDB.

King Burger

Originally called Eat-Da-Burger, King Burger was created as a way to work with a database to dynamically render text based on user entry. This app also uses a home-made ORM as a part of learning to work using the MVC design scheme.

Beerfinder 2.0

There are SO many apps on the internet the help people discover what types of wine they might enjoy, but craft beer wasn't getting the same love. This collaborative project was born in response. Using Sequelize as an ORM for this MVC-structured application, we designed a survey that generates a recommendation based on 8 popular styles of beer in the United States.
